# AI/ML project: Rule Based BMI Calculator
A diagnostic tool designed to categorize individuals based on their physical metrics using rule-based artificial intelligence logic.
---
## Project Overview
This project addresses the problem of health categorization by automating the Body Mass Index (BMI) calculation and classification process. It acts as a deterministic expert system, mapping numerical inputs( Weight and Height) to discrete health categories (Normal Weight, Obese, Undderweight, Overweight) based on World Health Organization (WHO) standards.
## Dependencies Installation
* **External Libraries:** None
* **Standard Libraries:** This project uses only built-in python modules to ensure maximum compatibility and zero-footprint execution.
* No 'pip install' commands are required for this project.
## Configuration steps
No complex confirguration or environment variables are needed. The script is self-contained and ready for immediate execeution.
## Execution Steps (CLI)
This project must be run via Terminal or Command Prompt. **It does not require a GUI**
1. Open your terminal or command prompt.
2. Run the script using:
   python main.py
3. **Interactions:**
   * Enter your weight when prompted.
   * Enter your height in cm when prompted.
   * The system will automatically process the units and display your BMI score and health category.
